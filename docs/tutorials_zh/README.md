# Prajna 教程索引

面向“每个知识点一个文档”的系列教程。建议从上到下阅读，或按需跳转。

- 01 运行与模式: 运行 Main、脚本、REPL、Jupyter（快速体验）
- 02 变量与类型系统: 基本类型、字面量、强/静态类型
- 03 函数、方法与闭包: 函数定义、返回值、闭包与捕获
- 04 结构体、接口与动态分发: struct、implement、interface、Dynamic
- 05 模板与泛型: template 的类型/常量参数、常见范式
- 06 指针与内存: ptr<T> 与 Ptr<T>、引用计数、生命周期
- 07 容器 I: Array、DynamicArray、List
- 08 容器 II: Dict/HashDict、Set、String、排序
- 09 张量与数值: Tensor、Layout、主机/显存拷贝
- 10 GPU 编程总览: 统一索引 API、NV/AMD 平台要点
- 11 C 库绑定与动态链接: @extern、#link、示例（ZMQ）
- 12 模块与包: use、命名空间、聚合入口 .prajna
- 13 IO 与时间: 输入、打印、chrono
- 14 序列化与测试: Serializable/Serialize、@test
- 15 性能优化、调试与 FAQ

对应文件见本目录的同名文档。
